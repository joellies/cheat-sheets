Sélecteurs

Nom de l'élément HTML sur lequel on veut appliquer la règle CSS. Pour en sélectionner plusieurs qui regroupent les mêmes propriétés, on utilise des ",".

Basiques

Documentation MDN

Sélecteur de type

  • div {}
  • span {}
  • p {}
  • a {}
  • table {}
  • etc.

Sélecteur de classe, d'identifiant et d'attributs

  • .important {}
  • #important {}
  • div .important {}
  • input[type="valeur"] {}
  • p #attribut * {}

Combinateurs

  • div + p {} permet de sélectionner les p qui suivent la div
  • p ~ span {} permet de sélectionner les span qui ont le même parent
  • ul > li {} permet de sélectionner tous les li enfants directs de ul
  • div p {} permet de sélectionner tous les descendants p (pas nécessairement directs) de div.
  • col || td {} permet de sélectionner n'importe quel élément td dans une colonne col.

Pseudo-éléments

Documentation MDN

Permet de mettre en forme seulement certains éléments du sélecteur.

Syntaxe : sélecteur ::pseudo-élément {propriété : valeur}; 

  • .class::after
  • #id::before
  • p::first-letter
  • p::first-line
  • ::placeholder

Pseudo-classes

Documentation MDN

Permet de sélectionner les éléments selon l'état que l'on souhaite attribuer (quand on passe la souris dessus par exemple). 

  • a:active
    cible un lien lorsque celui-ci est activé par l'user 
  • a:focus
    cible un élément lorsque celui-ci reçoit le focus (champ d'un formulaire) 
  • a:hover
    spécifie l'apparence d'un élément au moment où l'user le survole avec le pointeur 
  • a:link
    modifie l'aspect d'un lien tant qu'il n'a pas été cliqué. 
  • a:visited
    modifie l'aspect d'un lien après que l'user l'a visité 
  • :first-child
    cible un élément qui est le premier élément enfant par rapport à son élément parent 
  • :first-of-type
    cible le premier élément d'un type donné parmi ceux d'un même élément parent (et de même niveau) 
  • :last-child
    cible un élément qui est le dernier enfant de son parent 
  • :last-of-type
    sélectionne le dernier élément qui apparaît 
  • :nth-child(an+b)
    sélectionne un élément par rapport à sa position dans un groupe d'élément semblable (permet de rajouter even et odd) 
  • :nth-last-child(an+b)
    idem mais en partant de la fin 
  • :nth-of-type (an+b)
    correspond à des éléments d'un type donné, en fonction de leur position au sein d'un groupe de frères et sœurs

Flexbox

Documentation Flexbox et online builder

Propriétés

  • flex-direction : row | row-reverse | column | column-reverse
  • flex-wrap : wrap | no-wrap | wrap-reverse
  • flex-flow : flex-direction flex-wrap
  • justify-content (axe principal) : flex-start | flex-end | center | space-between | space-around | space-evenly
  • align-items (axe secondaire) : flex-start | flex-end | center | baseline | stretch
  • order (1 en bas, 10 au-dessus)
  • flex-grow : s’étire dans l’espace restant
  • flex-shrink : se contracte si nécessaire
  • flex-basis : taille initiale de l’élément avant que l’espace restant ne soit distribué
  • flex : grow shrink basis
