Variables

Le nom d'une variable est unique et claire. On utilise le camelCase pour la définir. Le = assigne la variable à une valeur : une chaîne de caractères (""), un nombre, un booléen (true/false), un array (tableau), un objet (avec des propriétés et des valeurs). 

  • const myName = "Joéllie"

    • déclarer une variable qui ne contiendra qu'une valeur qui ne pourra être accessible seulement si elle précédemment déclarée.

  • let myAge = 29

    • déclarer une variable, mais sans la contrainte d'assignation unique.

Fonctions

La documentation sur la MDN

Une fonction est un ensemble d'instructions menant à la réalisation d'une tâche, c'est un bloc de code réutilisable. Elle contient un constructeur () qui peut recevoir un ou plusieurs paramètres (,) et les instructions à executer englobées via {}.

(param1, param2, …) => {
          return expression;
        }

On peut la déclarer via le mot-clé function ou via la fat arrow function.

([param] [, param]) => {
          instructions
       }

Une fonction sans paramètres

() => {
          instructions
        }
Il ne faut pas oublier d'appeler la fonction

nom()

                    })

                      })
                      .then(response => {
                        console.log(response);

                          firstTen.push(item);
                          console.log(firstTen);
                        });

                          let currentDiv = document.getElementById('o-fetch__getExemple');
                          let newDivAuteur = document.createElement('div');

                          newDivComment.appendChild(newContentComment);
                          currentDiv.appendChild(newDivAuteur);
                          currentDiv.appendChild(newDivComment);

                          newDivComment.setAttribute('class', 'commentForm');
